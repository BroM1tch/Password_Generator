# Password Generator (CLI)

Secure password generator written in Python. Uses the `secrets` module for cryptographic randomness.

## Features
- Generate secure passwords with configurable length
- Enable/disable character sets:
  - Uppercase letters
  - Digits
  - Symbols
- Generate multiple passwords at once (`--count`)
- Friendly error messages for invalid input

## Requirements
- Python 3.10+

## How to run
From the project root:

```bash
python src/main.py
